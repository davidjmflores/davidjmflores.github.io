Tabular SARSA for Gridworld Maze Navigation

Home Projects GitHub

Overview

This project implements a tabular SARSA agent in a custom gridworld maze. The agent learns a policy for navigating from a start state to a goal state while avoiding wall cells.

Environment

  • State space: discrete grid cells
  • Action space: up, down, left, right
  • Obstacles: wall cells
  • Goal: terminal state

Algorithm

SARSA is an on-policy temporal-difference control method. The agent updates action values using the current state-action pair, reward, next state, and next action selected by the same behavior policy.

Results

Training plots, learned policy visualization, and episode performance will be added as the experiments are finalized.
