Overview

Reinforcement learning (RL) is often introduced through simple environments, but applying it to robotics requires dealing with larger state spaces, noisy observations, continuous motion, and physical constraints. This project is meant to build that bridge gradually from toy examples to high fidelity simulations. The goal is to study how different RL methods behave in controlled navigation problems, then use those results to guide later work in continuous and sensor-based robotic environments. The project begins with discrete gridworlds because they make algorithm behavior easy to inspect and compare.

Project Roadmap

Phase 1: Discrete 2D Gridworld

The first phase builds a custom tabular gridworld framework with configurable maps, obstacle layouts, start and goal states, and a visualizer for inspecting agent behavior. This phase is used to compare classical tabular RL algorithms under controlled navigation tasks.

  • Custom discrete gridworld environment
  • Gridworld visualizer
  • Multiple map layouts
  • Tabular SARSA, Expected SARSA, n-step SARSA
  • Q-learning and Double Q-learning
  • Dyna-Q and Prioritized Sweeping
  • Hyperparameter sweeps over epsilon, alpha, gamma, and n
  • Final comparison between the best algorithm from each method family

Phase 2: Continuous 2D Navigation

The second phase will move from discrete grid cells to continuous 2D state spaces. This phase will introduce approximate RL methods, feature-based value functions, and simple simulated sensors for local navigation.

Phase 3: 3D Physics-Based Navigation

The final planned phase will extend the project into a 3D physics-based robotics environment with more realistic dynamics, improved sensors, and approximate or deep RL methods for navigation.
